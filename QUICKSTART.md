# Quick Start Guide

## Installation (First Time Only)

```bash
npm install
```

## Running the App

### On Your Phone (Recommended)
1. Install "Expo Go" app from App Store or Google Play
2. Run: `npm start`
3. Scan the QR code with your phone's camera (iOS) or Expo Go app (Android)

### On Web Browser
```bash
npm run web
```

### On Android Emulator
```bash
npm run android
```

### On iOS Simulator (Mac only)
```bash
npm run ios
```

## Optional: Add Success Sound
1. Download a short MP3 sound from https://pixabay.com/sound-effects/
2. Save as `assets/success.mp3`
3. App works without it but won't play sound

## Troubleshooting
- QR code not working? → Make sure phone and computer are on same Wi-Fi
- Errors on install? → Make sure you have Node.js 16+
- Need help? → Check README.md for detailed instructions
