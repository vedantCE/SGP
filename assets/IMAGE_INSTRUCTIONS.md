# Asset Placeholders

The Expo app expects the following image assets in the `assets` folder:

- `icon.png` - App icon (1024x1024px recommended)
- `splash.png` - Splash screen (similar to icon but can be rectangular)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px or larger)

## How to Create These:

### Option 1: Use Expo Defaults
Expo CLI will generate default placeholder images if these don't exist. The app will work fine with defaults.

### Option 2: Create Custom Assets
1. Create simple colorful images with:
   - App name "Trace & Learn"
   - Educational theme (pencils, letters, numbers)
   - Bright, child-friendly colors

### Option 3: Use Online Tools
- Use Canva or similar tools
- Search for "app icon generator" or "splash screen generator"
- Export in PNG format

## Important Note:
The app will still work without custom assets. Expo will use default placeholders.
